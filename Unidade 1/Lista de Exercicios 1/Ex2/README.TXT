Lista de exercícios

2. Considere um sistema para uma livraria e construa uma classe que represente os livros
vendidos na loja. Todo livro possui como atributos nome, autor, preço de custo, edição e
ano. Esta classe deve possuir os seguintes métodos:
- método para criar um livro com todos os seus dados.
- Construa um método para calcular o preço de venda do livro. Para isso deve ser informado
o percentual de lucro da loja a ser aplicado sobre o preço de custo do livro.
- Construa um método para imprimir os dados de um livro
